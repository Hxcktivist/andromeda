## Commands
*  `/free` <em>list all available commands</em>
*  `/free install` <em>open install panel</em>
*  `/free config` <em>open config panel</em>
*  `/free unlock` <em>unlock UI elements</em>
*  `/free reset` <em>reset all saved options data</em>
